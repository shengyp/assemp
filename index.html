3.1 Algebra Example

Here is the function $ f(x)=sin(x)-2\sqrt{3}*sin(\frac{x}{2})^{2} $.
(1) Get the minimum cycle of function $ f(x) $.
(2) Get the minimum value of function $ f(x) $ at $ [0,\frac{2π}{3}] $.

After being processed by NLP, we can get following relations.

Question Relations:

NO. Relations
1 CommonQustionRelations:
2 CommonFunctionRelation{function:$ f(x)=sin(x)-2\sqrt{3}*sin(\frac{x}{2})^{2} $,definitionDomain:R}
3 SubstemRelation:
4 (1) None
5 (2)$ x∈[0,\frac{2π}{3}] $
6 SubConclusionRelation：
7 (1) [SolutionConclusionRelation{relation:getMinClycle，CommonFunctionRelation{function:
$ f(x)=sin(x)-2\sqrt{3}*sin(\frac{x}{2})^{2} $,definitionDomain:R}}]
8 (2) [SolutionConclusionRelation{relation:getMinValue，CommonFunctionRelation{function:
$ f(x)=sin(x)-2\sqrt{3}*sin(\frac{x}{2})^{2} $,definitionDomain:R}}]

And,here is the solving result which contains a very detailed process of solving problems.

AutoSolving Result:

NO. Solving Result
1
(1)∵ $ f(x)=sin(x)-2\sqrt{3}*sin(\frac{x}{2})^{2} $
(2)∴ we can get it from (1): $ sin(x)-2\sqrt{3}*sin(\frac{x}{2})^{2} $
(3)∴ we can get it from (2): $ sin(x)-2\sqrt{3}+2\sqrt{3}*cos(\frac{x}{2}) $
(4)∴ we can get it from (3): $ sin(x)-\sqrt{3}+\sqrt{3}*cos(x) $
(5)∴ we can get it from (4): $ 2*sin(x-\frac{π}{3})-\sqrt{3} $
(6)∴ we can get it from (1) and (5): $ f(x) = 2*sin(x-\frac{π}{3})-\sqrt{3} $
(7)∴ so,we can get the min cycle from (6): $ 2π $ is the min cycle of the function $ f(x) = 2*sin(x-\frac{π}{3})-\sqrt{3} $.
2
(1)∵ $ f(x)=sin(x)-2\sqrt{3}*sin(\frac{x}{2})^{2} $
(2)∴ we can get it from (1): $ sin(x)-2\sqrt{3}*sin(\frac{x}{2})^{2} $
(3)∴ we can get it from (2): $ sin(x)-2\sqrt{3}+2\sqrt{3}*cos(\frac{x}{2}) $
(4)∴ we can get it from (3): $ sin(x)-\sqrt{3}+\sqrt{3}*cos(x) $
(5)∴ we can get it from (4): $ 2*sin(x-\frac{π}{3})-\sqrt{3} $
(6)∴ we can get it from (1) and (5): $ f(x) = 2*sin(x-\frac{π}{3})-\sqrt{3} $
(7)∵ $ x∈[0,\frac{2π}{3}] $
(8)∴ we can get it from (6) and (7): $ x+\frac{π}{3}∈[\frac{π}{3},π] $
(9)∴ we can get it from (8):$ 2sin(x-\frac{π}{3})-\sqrt{3}∈[-\sqrt{3},2-\sqrt{3}] $
(10)∴so, we can get the min value from (9):The minimum value of $ f(x) = 2*sin(x-\frac{π}{3})-\sqrt{3} $ at $ x=\frac{2π}{3} $.
The above examples are presented in terms of natural language understanding and human-like problem solving. First, we enter the question number to get the specific topic content. Secondly, the natural language processing module analyzes and understands the topics and conclusions and translates them into corresponding knowledge relationships. This forms the three relationships of CommonQustionRelations, SubstemRelation and SubConclusionRelation in the Question Relations table, each of which contains one or more specific knowledge relationships. Then, the reasoning module receives the result of the natural language processing, and inserts the common problem, the sub-question and the sub-conclusion relationship into the knowledge base for reasoning. We can see from the above AutoSolving Result table that the solution process is based on a strict logical relationship step by step. Finally stop the reasoning and output the result. This forms a complete solution process.
